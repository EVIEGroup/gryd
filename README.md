# Gryd
 
## Setup

Requires NODE >= 16.*

`npm install`

## Commands

### Run Tests

`npm run test`

### Debug Test (Node CLI - Watch file changes)
`npm run dev`

### Build & Run (Node CLI)

`npm run build`
`npm run start`

### Build (Webpack SDK)

`npm run web:build`

### Debug Test (Webpack SDK)
`npm run web`
